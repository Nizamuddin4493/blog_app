## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need to install ruby and rails in your computer. the following the cammand should run in CMD.

```sh
 gem install rails
```

### Setup

Clone this repository to your desired folder:

```sh
  git clone git@github.com:Nizamuddin4493/blog_app.git
  cd blog_app
  gem install
```

### Usage

To run the project, execute the following command:

```sh
  rails server
```

### Run tests

To run tests, run the following command:

```sh
  bin/rails test test/models/article_test.rb
```
